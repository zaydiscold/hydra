### 🔴 CRITICAL (2)
1. **Orphaned migration lock → permanent schema sync failure** — `schemaSync.js:63-101`. Lock file persists if crash during self-heal. No TTL, no PID check, no recovery. DB stuck forever. **[FIXED: commit 8e43627 — F1]**
2. **Quit button uses window.close() instead of quitApp()** — `App.jsx:623`. In Electron, bypasses full shutdown chain (Playwright cleanup, server shutdown, child kill). [Agent #6]

### 🟠 HIGH (12)

4. **before-quit preventDefault race** — `main.js:263-271`. Second quit signal skips preventDefault, force-kills mid-shutdown. [Agent #1]
