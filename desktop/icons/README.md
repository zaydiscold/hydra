# desktop/icons/ — Hydra App Icons

This directory contains the platform-specific application icons generated by
`scripts/generate-icons.mjs`.

## Files

| File       | Platform | Format          | Notes                               |
|------------|----------|-----------------|-------------------------------------|
| icon.png   | Linux    | PNG 512×512     | Used by AppImage, deb, rpm targets  |
| icon.ico   | Windows  | ICO (multi-res) | 16,32,48,64,128,256px embedded      |
| icon.icns  | macOS    | ICNS (multi-res)| 16…512@2x, requires macOS to build  |

## Regenerating

```bash
node scripts/generate-icons.mjs
```

This reads the source image from `public/hydra_dragon.png` (1024×1024) and
produces all three platform icons.

For the checked-in Hydra icon source, regenerate the source PNG, favicon, and
all platform icons together:

```bash
npm run icons:generate
```

**macOS requirement**: The `.icns` file can only be generated on macOS because it
relies on the native `iconutil` tool. On other platforms, the script skips `.icns`
generation with a warning.

To change the source image:

```bash
node scripts/generate-icons.mjs --source path/to/new-source.png
```

## Source

The source image is `public/hydra_dragon.png` — a generated 1024×1024 neon
Hydra badge with a readable `H` mark, three hydra-head silhouettes, and the
cyan/magenta app palette on a transparent rounded-square canvas. The source SVG
is generated by `scripts/generate-hydra-source-icon.mjs` and also written to
`public/favicon.svg`.
